NEXT STEPS:
============================================================

1. CREATE FIREBASE PROJECT
   - Go to: https://console.firebase.google.com/
   - Click "Create project"
   - Name it "SoulVoyage" (or your preferred name)
   - Accept terms and create

2. GET YOUR API KEY AND CONFIG
   - Go to Project Settings (gear icon) â†’ General
   - Scroll to "Your apps" section
   - Click Web app (</> icon)
   - Copy all 6 values:
     * apiKey
     * authDomain
     * projectId
     * storageBucket
     * messagingSenderId
     * appId

3. ADD VALUES TO .env FILE
   - Open: SoulVoyage/.env
   - Fill in your 6 Firebase values:
   
     REACT_APP_FIREBASE_API_KEY=YOUR_API_KEY
     REACT_APP_FIREBASE_AUTH_DOMAIN=YOUR_AUTH_DOMAIN
     REACT_APP_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
     REACT_APP_FIREBASE_STORAGE_BUCKET=YOUR_STORAGE_BUCKET
     REACT_APP_FIREBASE_MESSAGING_SENDER_ID=YOUR_MESSAGING_SENDER_ID
     REACT_APP_FIREBASE_APP_ID=YOUR_APP_ID

4. ENABLE AUTHENTICATION METHODS
   - In Firebase Console go to: Authentication â†’ Sign-in method
   - Enable "Email/Password" (both options)
   - Enable "Google"
   - Enable "Anonymous"
   - Click Save

5. TEST IN YOUR APP
   - Run: npm run dev
   - Go to: http://localhost:8081/login-auth
   - Test all 3 authentication methods
